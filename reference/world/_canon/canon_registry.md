# Canon Registry - VbN World Overview Builder

## Purpose
This registry tracks frozen canon content to prevent silent modification during the overview building process.

## Canon Freeze Rules
- Frozen files MUST NOT be altered
- New information may only be appended in new sections OR logged as Proposed Canon Additions
- Any contradiction with frozen canon must be logged as Canon Conflict (Frozen)
- Conflicts REQUIRE a checkpoint pause and user approval
