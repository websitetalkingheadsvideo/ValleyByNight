# MET Shining Host Player's Guide

This supplement for Mind's Eye Theatre Changeling: The Dreaming provides additional material for players of The Shining Host. The book expands on character creation options, provides new kiths, Arts, Realms, and other player resources for enhancing changeling characters and gameplay.

## Purpose and Scope

- This book is a supplement for The Shining Host, providing additional player resources.
- It expands character creation options and provides new material for changeling characters.
- Focuses on player-facing content rather than Storyteller material.
- Provides new kiths, Arts, Realms, and other character options.

## Additional Character Options

The book provides expanded character creation options, including:
- New kiths and kith variations
- Additional character concepts
- Expanded backgrounds and merits
- New character archetypes

## New Arts and Realms

Additional magical powers and realms of reality that changelings can access:
- New Arts for changelings to learn
- Additional Realms for use with Arts
- Expanded magical options

## Kith Information

Detailed information on various kiths, including:
- Kith characteristics and abilities
- Kith culture and society
- How different kiths relate to each other
- Roleplaying different kiths

## Character Development

Guidance on developing changeling characters, including:
- Advancing characters
- Learning new Arts
- Developing relationships with other changelings
- Exploring character concepts

## Key Themes

- **Expanded Options**: The book provides more choices for players creating and developing changeling characters.
- **Character Depth**: Additional material for creating more complex and interesting changeling characters.
- **Player Resources**: Focus on material useful to players rather than Storytellers.

## Purpose and Value

MET Shining Host Player's Guide serves as:
- A supplement providing additional player resources for The Shining Host
- Expanded character creation and development options
- New kiths, Arts, and Realms for changeling characters
- Player-focused material to enhance changeling gameplay

This book is useful for players of The Shining Host who want more options for creating and developing their changeling characters, providing additional kiths, Arts, Realms, and other character resources.
