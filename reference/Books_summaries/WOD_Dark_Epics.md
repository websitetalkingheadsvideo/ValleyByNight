# World of Darkness: Dark Epics

The document is a comprehensive guide for running large-scale live-action roleplaying games in the World of Darkness, providing practical advice, systems, and tools for Storytellers managing games with 75 or more players, network games spanning multiple chronicles, and convention events.

## Purpose and Scope

- This book provides guidance for running large-scale LARP events in the World of Darkness.
- It addresses the unique challenges of games with 75+ players, network games, and convention events.
- It offers practical solutions for logistics, organization, and storytelling at scale.
- The book is designed for experienced Storytellers who want to expand their games or create network organizations.

## Introduction

The book begins by explaining the differences between small tabletop games and large live-action games. In a tabletop game, the Storyteller represents all non-player characters, but in a LARP, players interact directly with each other, creating a more immersive but complex experience.

### Scale and Scope

The book defines different scales of games:
- **Small Games**: 7-20 players, can be run like tabletop games
- **Medium Games**: 20-75 players, require more organization
- **Large Games**: 75+ players, need specialized systems and structure
- **Network Games**: Multiple interconnected chronicles
- **Convention Games**: Hundreds or thousands of players at events

## Chapter One: Introduction

Establishes the foundation for understanding large-scale LARPing, including the differences between small and large games, the challenges of scale, and the rewards of running successful large games.

### Consistency and Resources

The book emphasizes the importance of consistency when running network games or large events. Sourcebooks provide a shared foundation that ensures all chronicles remain "on the same page" regarding the World of Darkness setting.

### Durability

Addresses how Mind's Eye Theatre books are designed for short- to medium-term chronicles, but can be adapted for long-term play. Discusses character growth rates and how to manage advancement in large or network games.

## Key Concepts

### Large Games

Games with 75+ regular players require:
- Larger venues
- More complex plots
- Better organization
- Multiple Storytellers/Narrators
- Clear communication systems

### Network Games

Interconnected chronicles that share:
- Consistent setting information
- Coordinated metaplot
- Character travel between games
- Shared rules and policies
- Administrative structure

### Convention Games

Large one-time or annual events featuring:
- Hundreds or thousands of players
- Multiple simultaneous plots
- Special event coordination
- Unique challenges and opportunities

World of Darkness: Dark Epics serves as:

- A comprehensive guide for running large-scale LARP events
- Practical advice for expanding small games into large ones

- Reference material for experienced Storytellers taking on bigger challenges

This book is essential for any Storyteller wanting to run large games, create network organizations, or manage convention events in the World of Darkness, providing the tools and knowledge needed to successfully scale up live-action roleplaying experiences.
