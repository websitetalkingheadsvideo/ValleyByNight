## Clanbook Structure

Each clanbook includes the following sections:

1. **Overview (Phoenix Lens)** - How the clan operates specifically in Phoenix
2. **Territory & Feeding** - Primary territories, feeding risks and advantages
3. **Local Power Web** - Allies, rivals, and institutional influence
4. **Themes & Touchstones** - Phoenix-specific themes and narrative elements
5. **Chronicle Hooks** - 6-10 story hooks for integrating the clan into chronicles
6. **NPC Hooks & Seeds** - At least 6 detailed NPC concepts with:
   - Concept/Role
   - Domain/Neighborhood
   - Clan tie-in
   - Motivation
   - Conflict
   - Secret
   - Connection prompt
7. **Quick Reference** - Key locations, themes, NPCs, and story hooks

## Using Clanbooks for NPC Creation

Each clanbook's **NPC Hooks & Seeds** section provides ready-to-use NPC concepts that can be:

- **Adapted into Character Teasers**: Use the NPC hooks as starting points for creating full Character Teaser files (see [NPC Character Teaser Tracking System](../Scenes/Character%20Teasers/tracking/README.md))
- **Developed into Database NPCs**: Import NPC concepts into the database using the [Character Import Guide](../game-lore/IMPORT_GUIDE.md)
- **Integrated into Chronicles**: Use the NPC hooks as seeds for developing ongoing storylines and character relationships

### Example Workflow

1. **Choose a Clan**: Select a clan relevant to your chronicle or story needs
2. **Review NPC Hooks**: Browse the NPC Hooks & Seeds section for concepts that fit your story
3. **Develop the NPC**: Expand the hook into a full character using the provided details (concept, domain, motivation, conflict, secret)
4. **Create Character Teaser**: Use the connection prompt as inspiration for how the NPC relates to the coterie
5. **Integrate into Database**: Add the NPC to the database and create the corresponding Character Teaser file

## Phoenix Setting Context

All clanbooks are set in **Phoenix, Arizona, 1994**, during the aftermath of the Prince's murder. This setting provides:

- **Political Chaos**: The Camarilla is leaderless, with factions scrambling for power
- **Territorial Conflicts**: Anarchs, Sabbat, and various clans compete for control
- **Unique Geography**: Desert surroundings, urban sprawl, and specific neighborhoods shape clan operations
- **Specific Locations**: Real Phoenix locations (24th Street, Scottsdale, Mesa, etc.) inform territory and feeding grounds

For more setting details, see [Setting Overview](../game-lore/Setting.txt).
