# First Impression — System Summary (VbN)

First Impression is a single, qualitative tag created the first time a PC meaningfully interacts with an NPC. It represents how the NPC frames the PC socially, not trust, liking, or loyalty.

## Key Rules

- One tag per PC–NPC pair
- Created on first interaction only
- No stacking; only replaced
- Biases tone and tolerance, not plot branches
- Exists before (and alongside) relationship score

## Canonical Tag Set (final)

- **pleasant** — polite, safe, socially competent
- **intriguing** — interesting, unusual, worth engaging
- **boring** — forgettable but inoffensive
- **awkward** — rough but not hostile
- **prickly** — dismissive or abrasive
- **unsafe** — threatening, disrespectful, or Masquerade-risky

## Lifecycle

- **Creation**: Assigned based on first dialog reaction intent + tone
- **Persistence**: Carries across scenes/nights, colors NPC reactions
- **Override**: Replaced only by:
  - relationship score thresholds, or
  - major social events (boon, public insult/defense, Masquerade incident)

## Mechanical Role

- Modifies NPC reaction bias and tolerance
- Affects opening lines and forgiveness
- Does not unlock quests, boons, or status

## Design Principle

Players express freely → NPCs react predictably → Consequences accumulate slowly
