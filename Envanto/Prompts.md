# Item Image Generation Prompts

This document contains AI image generation prompts for items that failed to download from Envato Photos. All prompts follow the Valley by Night Art Bible (Part XI — Items System) standards.

## Prompt Format

All prompts must include:

### Mandatory Meta-Prompt Prefix
```
Use Valley by Night global aesthetic:
- gothic noir
- phoenix 1994
- desert modern
- low saturation
- atmospheric haze
- deep shadows with controlled highlights
- gold and crimson accents
- subtle supernatural undertones
```

### Mandatory Meta-Prompt Negative Block
```
(no neon, no bright white LED, no glossy plastic, no modern 2020s tech, no cartoon, no comedic tone, no oversaturation, no sci-fi, no cyberpunk, no pastel, no rainbow effects)
```

### Image Specifications
- **Resolution:** 1024×1024 (1:1 square format)
- **Style:** Noir, low saturation, strong shadows
- **Lighting:** Controlled, dramatic, noir-inspired
- **Background:** Dark, textured, or blurred environment
- **Composition:** Centered item, clear visibility of key features
- **Color Grading:** Muted palette, gold and crimson accents where appropriate
