Project Overview

Valley by Night (VbN) is a comprehensive web-based system for running and managing a Laws of the Night Revised chronicle. Built as a faithful digital adaptation of the Mind's Eye Theatre system, VbN provides storytellers and players with tools to create characters, manage the social and political landscape, track boons and status, and access the complete rules system through AI-powered agents.

The system is built on PHP, MySQL, HTML, CSS, and JavaScript, maintaining compatibility with the original tabletop rules while adding digital conveniences like automated rulebook search, character relationship tracking, and interactive location mapping.

Example Character: Cordelia Fairchild

Cordelia Fairchild
Toreador • 8th Generation • Harpy of Phoenix

Concept: Elegant manipulator and long-standing Harpy of Phoenix, a Toreador of terrifying poise and psychic depth.

Biography: Embraced in 1914 at the age of 28, Cordelia Fairchild was a socialite and art critic during the height of the American Gilded Age's decline. Her Embrace came from a Toreador enamored with her ability to destroy reputations with a single whispered review. For decades she moved through Arizona's small but decadent Kindred society, first establishing herself in Flagstaff, where she cultivated influence through the city's business elite and railroad-connected families. By the late 1940s, she had relocated to Phoenix, bringing her mortal and Kindred contacts with her. Her rise was inevitable: by the 1950s she was recognized as the Harpy of the domain, her wit and cruelty wrapped in honeyed civility. In 1960, she orchestrated the social destruction of Darrell Buyers, a Nosferatu elder who once controlled the city's mortal political machine. The scandal ended his influence and solidified Cordelia's supremacy. She remains the Harpy to this night, as elegant as she is terrifying, a living reminder that in the Jyhad of whispers, words kill faster than blades.

Technical Achievements

Valley by Night features a sophisticated agent-based architecture that exposes (makes accessible) the game content through intelligent, AI-powered systems. These agents handle everything from rulebook queries to character relationship analysis, making the rich game content easily accessible to storytellers and players.

  • 🧛 Laws Agent

    AI-powered rulebook search system with access to 31 PDF rulebooks (4,500+ pages) in a searchable database. Provides natural language queries about Laws of the Night Revised rules, with citations and context. Includes file-based knowledge system for Laws of the Night content with markdown indexing.
    Links: Laws Agent Interface | Rulebooks Database Documentation

  • 👤 Character Agent

    Monitors character JSON files for consistency, generates character briefs, suggests plot hooks, and performs continuity checks. Validates character data, checks sire relationships, generation consistency, and timeline conflicts. Generates daily and continuity reports. Provides searchable interface for character queries and analytical questions.
    Links: Character Search

  • 🤝 Boon Agent

    Monitors and validates boons according to Laws of the Night Revised mechanics. Tracks favor-debt relationships, detects violations, integrates with Harpy systems, and generates boon relationship graphs. Can generate character-specific boon networks with deterministic distribution algorithms.
    Links: Boon Ledger

  • ⚔️ Discipline Agent

    Provides searchable interface for all vampire disciplines with levels, powers, and mechanics. Displays discipline information in a clean, organized format with full power descriptions and level requirements.
    Links: Discipline Agent

  • 📚 Ability Agent

    Searchable interface for character abilities with sorting and filtering. Displays ability categories, levels, and specializations according to Laws of the Night Revised rules.
    Links: Ability Agent

  • 🛤️ Paths Agent

    Provides interface for viewing morality paths (Humanity, Path of Enlightenment, etc.) with path ratings, powers, and descriptions. Displays character path assignments and progression.
    Links: Paths Agent

  • 🔮 Rituals Agent

    Searchable interface for Thaumaturgy and Necromancy rituals with levels, requirements, and full descriptions. Tracks which characters know which rituals and provides ritual lookup functionality.
    Links: Rituals View

  • 🗺️ Map Agent

    Interactive map of Phoenix showing all locations with zoom, pan, and location details. Displays location types, districts, and allows filtering by various criteria. Provides visual representation of the game world.
    Links: Phoenix Map

  • 👥 Coterie Agent

    Manages coterie (group) information including members, roles, strengths, gaps, story hooks, and internal tensions. Tracks coterie focus and provides relationship mapping. Supports PC coterie creation and management.
    Links: Coterie Agent

  • 💼 Positions Agent

    Tracks Camarilla positions (Primogen, Harpy, Sheriff, etc.) with current holders, assignment history, and position details. Provides queries for position lookups and character position history.
    Links: Camarilla Positions

  • 📊 Influence Agent

    Provides lookup interface for Influence types and levels with effects, contacts, and allies information according to Laws of the Night rules. Displays influence effects by type and level.
    Links: Influence Agent

  • 🎨 Style Agent

    MCP-based system for accessing the Valley by Night Art Bible, providing visual style guidelines, asset specifications, and design standards for character portraits, locations, and cinematic content. Ensures visual consistency across all project assets.
    Links: Style Agent Documentation

Historical Context

Valley by Night began as a character creation system for Laws of the Night Revised, evolving into a comprehensive chronicle management platform. The project maintains strict adherence to the original Mind's Eye Theatre rules while adding digital conveniences that enhance gameplay without changing the core mechanics.

The system has been developed with input from experienced LARP storytellers and players, ensuring that digital tools complement rather than replace the tabletop experience. All agents and systems are designed to support storytellers in running complex, character-driven chronicles while maintaining the social and political dynamics that make Laws of the Night engaging.

The project follows a version-controlled development process with comprehensive documentation, ensuring that all changes maintain compatibility with Laws of the Night Revised rules and the established Valley by Night setting. Current version 0.9.15 represents over 100 version increments of iterative development and refinement.
