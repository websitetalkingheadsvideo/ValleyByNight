## Directory Structure Pattern

Each clan follows this standard structure:

```
[ClanName]/
├── index/
│   └── [clan]_section_index.json
├── mechanics/
│   └── [Clan]_Mechanics.json
├── npc_seeds/
│   └── [clan]_npc_seeds.json
├── phoenix/
│   └── [clan]_phoenix.json
│   └── [Clan]_Phoenix_Hooks.json (optional)
│   └── [Clan]_Phoenix_NPCs.json (optional)
│   └── [Clan]_Phoenix_Politics.json (optional)
│   └── [Clan]_Phoenix_Profile.md (optional)
├── profile/
│   └── [Clan]_Profile.json
└── tone_bible/
    └── [clan]_tone_bible.md
```

Some clans also have root-level markdown files (e.g., `[Clan].md`).
