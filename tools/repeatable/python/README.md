## Categories

### [JSON Tools](json-tools/README.md)

### [Text Tools](text-tools/README.md)
Tools for text processing and cleanup.

### [PDF Tools](pdf-tools/README.md)

## Common Dependencies

Most Python tools require:
- Python 3.7+
- Standard library modules (json, re, os, sys, pathlib, etc.)
- See individual tool README files for specific dependencies

Each tool category has its own README with detailed usage instructions. See category directories for specific tool documentation.
