# PART III — LOCATION & ARCHITECTURE SYSTEM  
## (Full, Exhaustive Version)

## 1. Overview
The Valley by Night Location & Architecture System defines the visual identity of every environment in the chronicle. It merges:
- Phoenix 1994 authenticity
- Gothic Noir cinematic tone
- Desert Modern architectural realism
- Clan and factional styles

All location art must integrate with:
- Cinematic system (Part II)
- UI palette (Part V)
- Floorplan system (Part VIII)
- 3D asset system (Part IV)

## 2. Global Environment Philosophy
Locations evoke:
- urban isolation
- desert stillness
- heat-haze atmosphere
- decaying neon
- faint menace

Never use:
- bright modern LED lighting
- futuristic architecture
- clean, sterile visuals
- crowded streets

## 3. Architectural Tri-Fusion Rule
All VbN environments fuse three styles:

### Phoenix 1994 Architecture
- stucco walls
- strip malls
- asphalt parking lots
- palm silhouettes
- low-rise commercial buildings
- boxy geometry

### Gothic Noir
- heavy shadows
- dramatic lighting
- stone, iron, velvet hints
- high-contrast silhouettes

### Desert Modern
- concrete + glass
- dust accumulation
- long straight corridors
- sparse vegetation

## 4. Color & Lighting
Locations must use the core palette:
- Gothic Black (#0d0606)
- Dusk Brown-Black (#1a0f0f)
- Blood Red (#8B0000)
- Desert Amber (#C87B3E)
- Teal Moonlight (#0B3C49)
- Parchment Light (#f5e6d3)

Lighting rules:
- warm interiors
- amber streetlights
- teal moonlit exteriors
- deep shadows
- soft haze

## 7. Camera & Composition for Location Art
- cinematic framing (rule of thirds)
- medium-wide or wide angles
- strong silhouettes
- deep focal contrast
