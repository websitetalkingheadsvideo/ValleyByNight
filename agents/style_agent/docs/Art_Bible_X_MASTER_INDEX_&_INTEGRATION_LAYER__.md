# 3. Cross‑System Integration Model

Every subsystem interacts with the others. This matrix defines how.

## 3.1 Portrait ↔ Cinematic Integration
- Portrait lighting matches cinematic rim/key directions  
- Character silhouette consistency between mediums  
- Wardrobe style continuous across portrait, cinematic, and 3D  

## 3.2 Location ↔ Cinematic Integration
- Every cinematic location must reference the architecture standards in Part III  
- Floorplans (Part VIII) define spatial layout for camera movement in Part II  
- Environment props from Part IV match scene needs  

## 3.3 UI ↔ Portrait Integration
- Portraits appear in modals, cards, and info panels  
- UI color palette derived from cinematic lighting palette  
- Modal frames match portrait frame motifs  

## 3.4 3D ↔ Cinematic Integration
- 3D characters use same lighting rules as cinematics  
- 3D environments align with location style guide  
- Animatics reference 3D blocking and camera paths  

## 3.5 Marketing ↔ Everything Else
Marketing art is the "heightened" composite of all systems:
- portraits (Part I)
- cinematic visuals (Part II)
- location palettes (Part III)
- dramatic lighting (Part V)
- symbolic objects (Part VII)

## 3.7 Items ↔ UI ↔ Character Integration
- Items displayed in admin panel follow Part V — UI & Web Art System standards
- Item images use same lighting and composition principles as portraits (Part I)
- Items assigned to characters appear in character view modals
- Item badges and colors integrate with UI color palette
- Special artifacts may reference cinematic visual language (Part II)

## 3.6 Blueprint ↔ Location ↔ Cinematic
- Floorplans define layout for in‑game navigation & cinematics  
- Locations use blueprint styles to stay consistent in 2D & 3D  
- Cinematics must not contradict blueprint spatial logic

# 5. Unified Aesthetic Manifesto

The Valley by Night aesthetic must always evoke:

### Gothic Noir  
- deep shadows  
- punctuated highlights  
- atmospheric smoke/haze  
- elegant yet decayed  

### Phoenix 1994  
- desert heat  
- old strip malls  
- sodium streetlights  
- urban emptiness  

### Psychological Tension  
- restraint  
- menace  
- emotional depth  
- haunted stillness  

### Subtle Supernatural  
- no spectacle  
- quiet dread  
- hints, not explosions  

### Cinematic Quality  
- film grain  
- vignette  
- careful composition  

This manifesto is the final arbiter for artistic decisions.

---

# 6. Master Negative Aesthetic Rules

Never allow the following:
- neon cyberpunk  
- bright LED lighting  
- glossy plastic surfaces  
- superhero poses  
- comedic exaggeration  
- cartoon shading  
- modern 2020s tech  
- Instagram-style editing  
- oversaturated colors  
- lens flares  
- flat lighting  
- generic fantasy elements  

If any asset violates this, it is rejected.
