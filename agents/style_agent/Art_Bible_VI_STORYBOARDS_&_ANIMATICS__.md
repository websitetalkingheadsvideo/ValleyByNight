# PART VI — STORYBOARDS & ANIMATICS  
## (Full, Exhaustive Version)

## 1. Overview
Storyboards and animatics serve as the previsualization backbone of all Valley by Night cinematic sequences.  
They define:
- shot composition
- camera flow
- emotional rhythm
- lighting direction
- scene blocking
- pacing for VO-driven noir cinematics

All boards must follow the established gothic–noir, Phoenix 1994 tone.  
They must harmonize with:
- Cinematic System (Part II)
- Location Styles (Part III)
- Lighting & Color (global palette)
- Character portrait aesthetic (Part I)

---

# 2. Rendering Style

## 2.1 Grayscale Only
Storyboards are **always grayscale**, never colored:
- charcoal or graphite look
- 80% dark values
- 15% mid-tone
- 5% white highlights

## 2.2 Texture & Lines
- Rough charcoal line weight  
- Thick outlines for silhouettes  
- Medium lines for props  
- Thin detail lines for faces or emotional beats  
- Minimal smooth gradients  

## 2.3 Atmosphere
- Dust haze  
- Light bloom from candles or streetlamps  
- Soft vignette  
- Mild grain  

---

# 3. Panel Specifications

## 3.1 Resolution
- **1920 × 1080** for every panel

## 3.2 Layout Options
### 6-Panel Layout
```
[1][2]
[3][4]
[5][6]
```

### 9-Panel Layout
```
[1][2][3]
[4][5][6]
[7][8][9]
```

Use 6-panel for character intros and 9-panel for complex narrative scenes.

# 4. Shot Types

### EST — Establishing
- tone setting  
- skyline, desert ridge, Elysium hall  

### WIDE
- environment context  
- sparse movement  

### MCU — Medium Close-Up
- preferred for dialogue  
- emotional beats  

### CU — Close-Up
- symbolic object  
- eye reaction  
- blood drop  

### OTS — Over the Shoulder
- confrontation  
- surveillance  
- tension shots  

### INSERT
- object or detail  
- goblet, lighter, tarot, photo  

### DUTCH (Malkavian only)
- 8–12 degree tilt  
- disorientation  

---

# 5. Camera & Motion

## 5.1 Motion Rules
- slow push-in  
- slow pan  
- slight zoom  
- minimal wobble (no handheld)  
- tracking used sparingly  

## 5.2 Forbidden Motions
- shaky cam  
- whip pans  
- quick cuts  
- frenetic movement  

---

# 6. Lighting & Tone

### Noir Lighting
- deep shadows  
- hard contrast  
- warm/cool interplay  
- precise spotlights in theater scenes  
- candlelight in Giovanni/Elysium areas  

### Institutional Lighting (Hospital)
- flickering fluorescence  
- offset shadows  

### Desert Lighting
- teal moonlight  
- dust air bloom  

---

# 7. Animatic Timing

## Scene Duration
- 30–60 seconds  
- Ideal: 45 seconds  

## Timing Guidelines
| Shot Type | Duration |
|-----------|----------|
| EST | 2–4 sec |
| MCU | 4–6 sec |
| CU | 2–3 sec |
| INSERT | 2–3 sec |
| Emotional beat | 4–6 sec |
| Supernatural moment | 2–4 sec |
| Final shot | 4–8 sec |
| Fade to black | 2–3 sec |

---

# 8. Audio in Animatics

## 8.1 Ambient Layers
- wind  
- AC hum  
- crowd hush  
- neon buzz  
- distant traffic  

## 8.2 Music
- low-string drones  
- minimal jazz  
- desert synth pads  
- faint choir for Giovanni  

## 8.3 SFX
Minimal but precise:
- glass clink  
- cloth movement  
- footsteps  
- old fluorescent hum  

## 8.4 Voice-Over
- 5–7 lines  
- introspective  
- poetic but grounded  
- always noir  

---

# 9. Storyboard Notation Standard

Each panel annotation includes:

```
Panel #: SHOT TYPE (Lens)
- Action description
- Camera movement arrow
- Lighting cue
- Audio note (optional)
- VO line number (optional)
```

Example:

```
Panel 4: MCU (50mm)
- Character turns slightly
- Camera push-in ←
- Candlelight warm right, shadow left
- VO Line 3
```

# 10. Pipeline

## 10.1 Steps
1. Create thumbnails  
2. Expand into full boards  
3. Assemble into 1080p animatic timeline  
4. Add basic motion  
5. Add ambient audio  
6. Add VO  
7. Export for cinematic team  

## 10.2 Tools
- Krita / Photoshop  
- Blender Grease Pencil  
- DaVinci Resolve / Premiere for animatics
