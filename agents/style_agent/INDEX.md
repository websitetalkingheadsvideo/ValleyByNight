## Usage Workflow

### For Art Generation
1. Start with **RULES.md** for quick constraints
2. Reference specific chapter in `docs/` for detailed rules
3. Use **PROMPTS.md** to find prompt templates
4. Check **INDEX.md** (this file) to locate relevant chapters

### For Style Validation
1. Check **RULES.md** for key constraints
2. Reference specific chapter for detailed requirements
3. Compare against clan-specific motifs
4. Verify technical specifications

### For Prompt Creation
1. Use **PROMPTS.md** for prompt structure
2. Reference chapter-specific prompts in `docs/`
3. Combine modules as needed
4. Include negative prompts

### For System Integration
1. Review **Art_Bible_X_MASTER_INDEX_&_INTEGRATION_LAYER__.md**
2. Check integration notes in relevant chapters
3. Ensure cross-system consistency
4. Reference unified aesthetic principles
